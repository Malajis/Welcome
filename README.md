# Welcome

## 介绍
Welcome 是一款专为欢迎新玩家设计的插件，适用于 Minecraft 服务器，旨在通过自定义欢迎消息和交互式按钮提升玩家体验。

## 功能
- 自定义欢迎消息
- 提供交互式按钮，便于新玩家快速了解服务器规则或加入社群
- 支持命令和 Tab 补全功能
- 简单易用的配置文件（`config.yml`）

## 使用说明

### 安装
1. 下载插件并将其放入服务器的 `plugins` 文件夹。
2. 启动服务器以生成配置文件。
3. 编辑 `plugins/Welcome/config.yml` 文件以自定义欢迎消息和按钮。
4. 重启服务器以应用更改。

### 配置
插件提供了一个配置文件 `config.yml`，您可以在此文件中修改以下内容：
- 欢迎消息内容
- 按钮显示文本
- 按钮对应的动作（如链接到服务器规则或 Discord 群组）

### 命令
- `/welcome reload` - 重新加载插件配置文件
